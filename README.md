Travel Planner App ✈️

這是一個專為旅行愛好者設計的應用程式，旨在幫助使用者高效、直觀地規劃每日行程。無論是景點排程、時間管理，還是行程總覽，此應用程式都能提供流暢的操作體驗，讓旅行規劃變得簡單有趣。
📸 功能預覽
1. 輕鬆新增行程

透過直觀的介面，使用者可以快速輸入景點名稱、時間與備註，輕鬆將每一個精彩時刻排入行程表。
2. 即時互動提示

內建友善的互動設計，當輸入資訊不完整時，系統會給予即時提醒，確保您的規劃邏輯無誤，避免遺漏重要細節。
3. 完整的行程總覽

規劃完成後，清晰的總覽頁面讓您能一眼掌握當日所有安排，並提供編輯與管理功能，隨時調整您的旅行計畫。
🚀 核心特色

    直覺式操作介面：專為提升規劃效率設計。

    錯誤預防機制：即時驗證功能，減少操作錯誤。

    清晰的視覺呈現：幫助使用者快速確認每日行程進度。
